# 版本管理规范 / Version Management

## 当前版本 / Current Version

**v1.03** — 查看完整更新记录：[CHANGELOG.md](CHANGELOG.md)

---

## 版本发布规则

1. **版本号变更（如 v1.03 → v1.04）必须经过许总监授权**
2. 任何功能添加、接口变更、定价调整均需提前确认
3. 提交 PR 时请注明涉及的版本变更内容

---

## 如何发布新版本

1. 更新 `VERSION` 文件中的版本号
2. 更新 `CHANGELOG.md`
3. 提交 PR，由许总监审核后合并
4. 合并后由许总监确认发布
